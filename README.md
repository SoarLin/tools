# Tools

這是一個使用 React (TypeScript) + Vite 開發的小型工具網頁。

## 開發

```bash
npm install
npm run dev
```

## 部署到 GitHub Pages

1. 確保已將程式碼推送到 GitHub 遠端倉庫。
2. 執行以下指令進行部署：

```bash
npm run deploy
```

這將會自動執行 `npm run build` 並將產生的 `dist` 資料夾內容推送到 `gh-pages` 分支。

3. 在 GitHub 儲存庫的 **Settings > Pages** 中，將 **Branch** 設置為 `gh-pages` 並儲存。
